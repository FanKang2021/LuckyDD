# 彩票数据分析与特征探索平台 PRD

## 1. 项目概述

### 1.1 项目名称
LuckyDD - 彩票数据分析与特征探索平台

### 1.2 项目定位
**仅供娱乐研究使用的个人数据探索工具**，帮助用户分析彩票开奖历史数据的统计特征，探索号码模式。

### 1.3 合规声明
> **免责声明**：本平台所有分析结果仅供娱乐研究，彩票开奖为随机事件，任何统计特征不构成投注建议。请勿将分析结果作为购买彩票的依据。

### 1.4 核心原则
- 仅使用公开可访问的数据
- 不实施任何非法手段获取数据
- 明确标注"娱乐用途"
- 尊重网站robots.txt协议

## 2. 功能需求

### 2.1 支持的彩种
| 彩种 | 玩法描述 | 数据来源 |
|------|----------|----------|
| 双色球 | 6红+1蓝 | 中彩网、新浪彩票 |
| 大乐透 | 5前区+2后区 | 中彩网、500彩票网 |
| 福彩3D | 3个数字 | 中彩网、500彩票网 |

### 2.2 核心功能模块

#### 2.2.1 数据采集模块
- 自动定时采集开奖历史数据
- 支持手动触发数据更新
- 数据去重和完整性校验
- 多数据源交叉验证

#### 2.2.2 数据展示模块
- 开奖结果列表（支持分页、筛选）
- 号码走势图（折线图、热力图）
- 历史数据导出（CSV/Excel）

#### 2.2.3 统计分析模块
- **基础统计**：冷热号、遗漏值、连号分析
- **区间分析**：号码分布区间、奇偶比例
- **和值分析**：和值范围、均值、标准差
- **趋势分析**：周期性特征、波动规律

#### 2.2.4 创新分析模块
- **模式挖掘**：频繁号码组合发现
- **异常检测**：识别异常开奖模式
- **时序特征**：时间序列分析、周期发现
- **关联规则**：号码间的关联关系挖掘

#### 2.2.5 智能推荐模块
- 基于统计特征生成推荐号码
- 推荐方案多样性评估
- 历史推荐命中率追踪

### 2.3 用户界面需求
- 响应式设计，支持移动端
- PWA支持，可安装为安卓应用
- 深色/浅色主题切换
- 数据可视化图表

## 4. 数据采集设计

### 4.1 数据源
| 网站 | 类型 | 数据内容 |
|------|------|----------|
| https://www.zhcw.com/ | 官方 | 开奖结果、历史数据 |
| https://kaijiang.500.com/ | 综合 | 开奖结果、专家分析 |
| https://lotto.sina.cn/ | 综合 | 开奖结果、数据分析 |

### 4.2 采集策略
- 定时采集：每天开奖后自动更新
- 增量更新：只获取最新数据
- 数据缓存：本地存储避免重复请求
- 请求限流：尊重目标网站，避免频繁请求

### 4.3 数据模型

#### 双色球数据模型
```typescript
interface DoubleColorBall {
  id: string;              // 期号
  date: string;            // 开奖日期
  redNumbers: number[];    // 红球号码 [6]
  blueNumber: number;      // 蓝球号码
  prizeLevel: {            // 奖项设置
    level1: number;        // 一等奖注数
    level2: number;        // 二等奖注数
    prizePool: string;     // 奖池金额
  };
}
```

#### 大乐透数据模型
```typescript
interface SuperLotto {
  id: string;              // 期号

  prizeLevel: {            // 奖项设置
```

## 5. 创新分析算法设计

### 5.1 模式挖掘算法
- **频繁项集挖掘**：发现高频号码组合
- **序列模式挖掘**：发现号码出现的顺序规律
- **聚类分析**：将相似的开奖结果分组

### 5.2 异常检测算法
- **孤立森林**：识别异常开奖模式
- **Z-score检测**：检测偏离均值的异常值
- **滑动窗口异常检测**：基于时间窗口的异常发现

### 5.3 时序分析算法
- **自相关分析**：发现周期特征
- **傅里叶变换**：频率域分析
- **LSTM预测**：基于深度学习的趋势预测

### 5.4 关联规则挖掘
- **Apriori算法**：发现号码间的关联关系
- **置信度计算**：评估关联规则的可靠性
